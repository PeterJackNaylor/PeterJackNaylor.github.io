- [PhD_template](https://github.com/PeterJackNaylor/my_phd_template) the template I used for my dissertation. (17/12/2019)
- [Cellular heatmap generation and classification](https://github.com/PeterJackNaylor/CellularHeatmaps). A project based on the segmentation of nuclei in histology images, from the segmentation we extract a certain number of features, perform an unsupervised dimension reduction on the nuclei sets, transform this extraction to the tissue in order to generate cellular heatmaps, and finally a supervised model for the classification of these heatmaps. (12/11/2019) (on going)
- [AutomaticWSI](https://github.com/PeterJackNaylor/AutomaticWSI) Prediction of the residual cancer burden from histology images prior to treatment, the code contains a full pipeline to analyse Whole Slide Images automatically. (8/12/2019) (on going)
- [BestNucleiModel](https://github.com/PeterJackNaylor/BestNucleiModel) is just a project where we perform a nested-cross validation distributed with nextflow to find the best nuclei segmentation model.  (10/12/2018)
- [segmentation_net package] (https://github.com/PeterJackNaylor/segmentation_net): appropriate python objects based on tensorflow for dealing with segmentation problems. (not available yet)
- [dynamic_watershed package](https://github.com/PeterJackNaylor/dynamic_watershed): we implement the splitting algorithm for splitting nuclei nucleas described in in 'Nuclei segmentation in histopathology images using deep neural networks'. (27/08/2018)
- [useful_wsi package](https://github.com/PeterJackNaylor/useful_wsi): a set of useful tools to deal with Whole Slide Image formats. (14/07/2018)
- [Segmentation of Nuclei in Histopathology Images by deep regression of the distance map](https://github.com/PeterJackNaylor/DRFNS):  this repository contains the code necessary in order to reproduce the work contained in the submitted paper. (24/04/2018)
